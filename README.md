`Python` `OpenAI API` `TF-IDF` `scikit-learn` `SQLite` `Ollama`

**92.6% accuracy · LLM fallback rate cut from 30.8% → 14.7%**

`Python` `LangChain` `ChromaDB` `OpenAI API` `ArXiv API`

**Plan → Reason → Act · Persistent dual-layer memory**
